# KiiK - Mobile payments everywhere

## Installation

If this gem actually talked to a service, you could use it in your Ruby
project. To do so, you would add this line to your application's Gemfile:

    gem 'kiik'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kiik

## Usage

```ruby
require 'kiik'

```

Enjoy!
